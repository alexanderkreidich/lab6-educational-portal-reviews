# Лабораторная работа №6. Образовательный портал - Отзывы

Веб-приложение "Образовательный портал" с функционалом отзывов к онлайн-курсам.

## Функционал

- Просмотр курсов по категориям
- Создание курсов (для авторизованных пользователей)
- **Система отзывов:**
  - Отображение 5 последних отзывов на странице курса
  - Страница всех отзывов с пагинацией
  - Сортировка: по новизне, сначала положительные, сначала отрицательные
  - Форма создания отзыва с оценкой (0-5 баллов)
  - Автоматическое обновление рейтинга курса

## Установка

```bash
# Создание виртуального окружения
python3 -m venv ve
. ve/bin/activate

# Установка зависимостей
pip install flask flask-login flask-migrate flask-sqlalchemy python-dotenv

# Применение миграций
cd app
flask db upgrade

# Создание пользователя
flask shell
>>> from models import db, User
>>> user = User(first_name='Иван', last_name='Иванов', login='user')
>>> user.set_password('qwerty')
>>> db.session.add(user)
>>> db.session.commit()
>>> exit()

# Запуск
flask run
```

## Данные для входа

- **Логин:** user
- **Пароль:** qwerty
